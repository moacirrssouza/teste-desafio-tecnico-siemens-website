# Teste Desafio Técnico Siemens Website

Este é um projeto de exemplo para o desafio técnico da Siemens, construído com React, Vite e TypeScript.

## Descrição

Este projeto parece ser uma aplicação web para gerenciar uma lista de livros. Ele inclui funcionalidades para listar, cadastrar, e possivelmente autenticar usuários.

## Tecnologias

As principais tecnologias utilizadas neste projeto são:

*   **React:** Uma biblioteca JavaScript para construir interfaces de usuário.
*   **Vite:** Um build tool moderno que oferece uma experiência de desenvolvimento rápida.
*   **TypeScript:** Um superset de JavaScript que adiciona tipagem estática.
*   **React Router:** Para roteamento no lado do cliente.
*   **Zustand:** Para gerenciamento de estado.
*   **Axios:** Para fazer requisições HTTP.
*   **Vitest:** Para testes unitários.

## Scripts Disponíveis

No diretório do projeto, você pode executar:

### `npm run dev`

Executa o aplicativo no modo de desenvolvimento.\
Abra [http://localhost:5173](http://localhost:5173) para visualizá-lo no navegador.

A página será recarregada se você fizer edições.\
Você também verá quaisquer erros de lint no console.

### `npm run build`

Compila o aplicativo para produção na pasta `dist`.\
Ele agrupa o React no modo de produção e otimiza a compilação para o melhor desempenho.

### `npm test`

Inicia o executor de testes no modo de observação interativo.

### `npm run lint`

Executa o linter para verificar erros de estilo de código.

### `npm run preview`

Inicia um servidor local para visualizar a compilação de produção.

## Como Executar

1.  Clone o repositório:
    ```sh
    git clone <url-do-repositorio>
    ```
2.  Navegue até o diretório do projeto:
    ```sh
    cd teste-desafio-tecnico-siemens-website
    ```
3.  Instale as dependências:
    ```sh
    npm install
    ```
4.  Inicie o servidor de desenvolvimento:
    ```sh
    npm run dev
    ```
